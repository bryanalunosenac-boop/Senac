# 🌀 Projeto - Metas diárias


O objetivo do projeto é criar metas com descrição e determinar a sua propriedade sendo elas (baixa, média e alta), com um input do tipo "Date" para adicionar um prazo de realização da meta prevista. E após o preenchimento do formulário aparecerá um campo que vai imprimir as informações listadas anteriormente.

---

## ⚙️ Funcionalidades

- Adicionar metas
- Remover metas
- Marcar como concluída
- Validação de campos
- Layout responsivo
